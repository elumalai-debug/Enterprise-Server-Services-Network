# Enterprise-Server-Services-Network
Enterprise multi-branch network simulation in Cisco Packet Tracer featuring Static Routing, DHCP, DHCP Relay, DNS, and HTTP Server services.

---


📌 Project Overview

This project demonstrates the design and implementation of a multi-branch enterprise network using **Cisco Packet Tracer**. The network connects four branch offices—**Chennai, Hyderabad, Pune, and Mumbai**—through Cisco routers using **static routing**. Various server services are configured to simulate a real-world enterprise network environment.

The project focuses on providing automatic IP address allocation, domain name resolution, web hosting, and seamless communication between different branch networks.

---

## 🚀 Features

* Multi-branch enterprise network topology
* Static Routing between all branch routers
* DHCP Server for automatic IP address assignment
* DHCP Relay Agent (IP Helper Address)
* DNS Server for domain name resolution
* HTTP (IIS) Web Server hosting custom webpages
* End-to-End connectivity between all locations
* Multiple LANs connected through WAN links
* Successful web access using domain names
* Network verification using Ping and Web Browser

## 🌍 Services Configured

### DHCP

* Automatic IP address allocation
* Dynamic subnet mask assignment
* Default gateway configuration
* DNS server assignment

### DHCP Relay

* Clients in remote networks receive IP addresses from the centralized DHCP server using **IP Helper Address**.

### DNS Server

* Domain name resolution
* Custom domain records (e.g., `www.azure.com`, `www.ccna.com`)

### HTTP (IIS) Server

* Custom enterprise webpages
* Web access using DNS names instead of IP addresses

### Static Routing

* Manual route configuration between branch routers
* Full communication between all branch networks
